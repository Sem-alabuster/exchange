CryptoEx (прототип)
==================

Файлы:
- index.html    — главная
- rules.html    — Правила / Политика конфиденциальности (читает тексты из .txt)
- rules.txt     — текст правил (не код)
- privacy.txt   — политика конфиденциальности (не код)
- logo.svg      — логотип
- START.bat     — запуск для Windows (сам поднимет сервер и откроет страницы)
- README.txt    — инструкция

Как открыть (самый простой способ в Windows):
1) Открой папку cryptoex_site
2) Дважды кликни START.bat

Как открыть вручную (если START.bat не подходит):
1) Открой папку cryptoex_site
2) Запусти локальный сервер:
   python -m http.server 5500
3) Открой в браузере:
   http://localhost:5500/index.html
   http://localhost:5500/rules.html

Почему так:
- Браузеры часто запрещают чтение rules.txt/privacy.txt при открытии файлов двойным кликом.
